# FPGA での各種演算の合成評価

加算/乗算などの速度と規模を実際に合成して確認するプロジェクト


| ディレクトリ名  | 説明                             |
| --------------- | -------------------------------- |
| mul_speed       | 乗算器の速度評価                 |
| mul_utilization | 乗算器のリソース消費評価         |
| add_speed       | 加算器の速度評価                 |
| mad_speed       | 積和器の速度評価                 |
| mux_speed       | マルチプレクサの速度評価         |
| mux_utilization | マルチプレクサのリソース消費評価 |
| counter_speed   | カウンタの速度評価               |
